# Спецификация: Облака и баблы (speech bubbles)

Дата: 2026-07-14
Статус: утверждено пользователем (дизайн), ожидает ревью спеки

## Цель

Добавить в редактор полноценные манга-баблы: диалоговые облака с хвостом,
облака мысли, крик, нарратив и шёпот. Бабл — единый объект «форма + хвост +
встроенный текст», который двигается целиком и участвует в общем порядке
слоёв, панели «Объекты» и экспорте.

## Утверждённые решения

- **Подход A**: новый тип объекта `BubbleObject` и коллекция `doc.bubbles`
  (не расширение `TextObject`).
- **Хвост**: перетаскиваемая точка-якорь на канве (любое направление и
  длина) + 8 кнопок-пресетов направления в панели как быстрый старт.
- **Текст внутри бабла**: контент, шрифт, размер, цвет — часть бабла;
  двигается вместе с ним, автоматически центрируется и переносится.
- **Размер**: авто по умолчанию (`autoSize: true`, бабл растёт под текст);
  ручной ресайз за рамку выключает авторазмер, текст переносится внутри.
- **Формы (полный набор)**: speech (овал), thought (бугристое облако +
  пузырьки-хвост), scream (зубчатый взрыв), narration (прямоугольник),
  whisper (пунктирный овал).

## Модель данных

`types/index.ts`:

```ts
interface BubbleObject {
  id: string;
  kind: 'speech' | 'thought' | 'scream' | 'narration' | 'whisper';
  visible: boolean;
  // Геометрия — нормированная к размеру документа (как texts/shapes)
  x: number; y: number;            // центр тела бабла
  width: number; height: number;   // размер тела
  rotation: number;
  autoSize: boolean;               // по умолчанию true
  tail: {
    enabled: boolean;
    tipX: number; tipY: number;    // кончик хвоста, нормированный
    width: number;                 // толщина у основания, доля от тела
  } | null;                        // narration создаётся с null
  fill: string;                    // по умолчанию #ffffff
  stroke: string;                  // по умолчанию #000000
  strokeWidth: number;
  text: {
    content: string;
    fontFamily: string;
    fontSize: number;              // в пикселях документа
    fill: string;
    align: 'left' | 'center' | 'right';
    lineHeight: number;
  };
}
```

`ImageDocument` получает `bubbles?: BubbleObject[]`. В единый порядок слоёв
(`utils/layerOrder.ts` / `resolveLayerOrder`) добавляется тип
`{ type: 'bubble', id }`; новый бабл добавляется НА ВЕРХ стека (как texts и
shapes). Undo/redo работает автоматически через существующий `pushHistory`
(баблы лежат внутри снапшота документа).

## Геометрия форм — общий модуль

Новый `utils/bubbleGeometry.ts` — единственный источник контуров. Функции
возвращают SVG path-строки (или массивы точек), используемые И канвой
(Konva.Path), И экспортом (Path2D на canvas 2D), чтобы превью и файл
совпадали пиксель в пиксель.

- `speech` — эллипс; хвост — клин от края тела к tip. Тело и хвост
  объединяются в один контур: заливка сплошная, обводка идёт вокруг
  объединённой фигуры (без внутренней линии на стыке).
- `thought` — контур из 10–14 дуг-бугров по эллипсу; хвост — 2–3
  убывающих отдельных пузырька, равномерно расставленных к tip.
- `scream` — зубчатая звезда: 16–24 луча с чередующимися внутренним и
  внешним радиусами и джиттером. Джиттер детерминирован сидом от `id`
  (не «дрожит» при перерисовках). Хвост — острый клин.
- `narration` — прямоугольник со скруглением 0–8 px; хвост опционален.
- `whisper` — эллипс, обводка пунктиром (`dash`); хвост — клин пунктиром.

Основание хвоста автоматически прилипает к точке пересечения контура тела
с лучом «центр → tip».

## Поведение на канве

Новый `components/editor/BubbleNode.tsx`, рендерится в общем цикле
`resolveLayerOrder` в CanvasArea:

- **Клик** — выделение (`selectedObject = { id, type: 'bubble' }`),
  Transformer: перемещение, ресайз, поворот. Ресайз выключает `autoSize`.
- **Двойной клик** — inline-редактирование текста: HTML `<textarea>`
  поверх бабла (позиционируется по координатам бабла с учётом зума),
  Escape или клик мимо — завершение. Учитывать CJK IME
  (`isComposing`). Текст также редактируется из панели настроек.
- **Хвост**: при выделении рисуется точка-якорь на `tip`; перетаскивание
  меняет `tipX/tipY`. Перемещение всего бабла двигает и tip (хвост
  сохраняет относительное положение).
- **Авторазмер**: при `autoSize` тело подгоняется под измеренный текст +
  паддинги; для thought и scream — множитель ~1.25, чтобы текст не
  задевал бугры/зубцы. При ручном размере текст переносится по ширине
  тела и центрируется вертикально.
- **Enter/Escape** снимают выделение (существующее глобальное поведение).

## Панель настроек

Секция «Баблы» во вкладке «Вставка» (`panels/InsertPanel.tsx` или новый
компонент `panels/BubblePanel.tsx`):

- 5 кнопок форм с мини-превью (SVG из bubbleGeometry).
- Переключатель хвоста + 8 кнопок-пресетов направления (N, NE, E, SE, S,
  SW, W, NW) — ставят tip на фиксированном расстоянии от тела.
- Слайдеры: толщина обводки, размер шрифта, межстрочный интервал.
- Цвета: заливка, обводка, текст.
- Шрифт: селект из уже загруженных в приложение шрифтов.
- Текстовое поле контента (синхронизировано с inline-редактором).

При выделенном бабле панель редактирует его; кнопки форм без выделения
создают новый бабл в центре видимой области.

## Интеграция

- **Панель «Объекты»** (RightPanel): строка с префиксом «B» и началом
  текста, через существующий универсальный механизм — drag-порядок,
  видимость, удаление.
- **Экспорт** (ExportModal): `drawBubble` рисует контуры из
  bubbleGeometry через Path2D + текст с переносами, в общем цикле
  `resolveLayerOrder`.
- **Store** (useStore): `addBubble`, `updateBubble`, `deleteBubble`,
  `duplicateBubble`; add ставит бабл на верх layerOrder.
- **Старые пресеты** «Манга-бабл» и «Крик» во вкладке «Текст» создают
  баблы нового типа (speech и scream) вместо текста с подложкой.
- **Автоперевод страницы** НЕ трогаем: распознанные реплики по-прежнему
  создают обычные тексты. Миграция на баблы — отдельный будущий блок.

## Ошибки и крайние случаи

- Пустой текст: бабл рисуется с минимальным размером (не схлопывается).
- tip внутри тела: хвост не рисуется (длина ≈ 0), без ошибок.
- Очень длинный текст при ручном размере: текст обрезается по телу
  (overflow hidden), пользователь видит и увеличивает бабл.
- Документы без поля `bubbles` (старые проекты): читается как `[]`.

## Критерии приёмки

1. Каждая из 5 форм создаётся, двигается, ресайзится, вращается.
2. Хвост тянется точкой на канве в любом направлении; 8 пресетов работают.
3. Двойной клик — редактирование текста на канве; текст центрируется
   и переносится; авторазмер растёт под текст.
4. Бабл виден в «Объектах», перетаскивается по порядку, скрывается,
   удаляется; undo/redo работает.
5. Экспортированный PNG/JPEG совпадает с превью (форма, хвост, текст).
